# Training Lab Templates

This folder is reserved for Training Lab template definitions and template-specific assets.

## Do not add unrelated assets

Do not add Loyalty Quest template images here unless David explicitly approves moving them into Training Lab.

## Template status labels

Use these labels in template docs or filenames where helpful:

- `final` — ready to use.
- `placeholder` — visual stand-in only.
- `mockup` — design reference only.
- `archive` — historical reference, not active.

## Required template notes

Each template should eventually document:

- Template name.
- Intended user role.
- Training goal.
- Required fields.
- Proof behavior.
- Reward behavior.
- Whether any database setup is required.
- Asset paths used by the template.
